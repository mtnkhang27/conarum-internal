# UI Components Reference

This file serves as a quick reference for the available UI components in this project.
AI Assistants: Read this file to know what components are available without scanning the entire directory.

## Core Components

| Component | File | Description |
| :--- | :--- | :--- |
| **Accordion** | `accordion.tsx` | A vertically stacked set of interactive headings that each reveal a section of content. |
| **Alert** | `alert.tsx` | Displays a callout for user attention. |
| **AlertDialog** | `alert-dialog.tsx` | A modal dialog that interrupts the user with important content and expects a response. |
| **AspectRatio** | `aspect-ratio.tsx` | Displays content within a desired ratio. |
| **Avatar** | `avatar.tsx` | An image element with a fallback for representing the user. |
| **Badge** | `badge.tsx` | Displays a badge or a component that looks like a badge. |
| **Breadcrumb** | `breadcrumb.tsx` | Displays the path to the current resource using a hierarchy of links. |
| **Button** | `button.tsx` | Displays a button or a component that looks like a button. |
| **Calendar** | `calendar.tsx` | A date field component that allows users to enter and edit date. |
| **Card** | `card.tsx` | A card component. |
| **Carousel** | `carousel.tsx` | A slideshow component for cycling through elements. |
| **Chart** | `chart.tsx` | Charting components. |
| **Checkbox** | `checkbox.tsx` | A control that allows the user to toggle between checked and not checked. |
| **Collapsible** | `collapsible.tsx` | An interactive component which expands/collapses a panel. |
| **Command** | `command.tsx` | Fast, composable, unstyled command menu for React. |
| **ContextMenu** | `context-menu.tsx` | Displays a menu to the user — such as a set of actions or functions — triggered by a button. |
| **DataTable** | `DataTable.tsx` | Advanced data table with sorting, filtering, and pagination. |
| **DataTableDialog** | `DataTableDialog.tsx` | A dialog wrapper for data tables. |
| **DatePicker** | `date-picker.tsx` | A date picker component. |
| **DateRangeFilter** | `date-range-filter.tsx` | A filter component for date ranges. |
| **Dialog** | `dialog.tsx` | A window overlaid on either the primary window or another dialog window. |
| **Drawer** | `drawer.tsx` | A drawer component. |
| **DropdownMenu** | `dropdown-menu.tsx` | Displays a menu to the user — such as a set of actions or functions — triggered by a button. |
| **Form** | `form.tsx` | Form wrapper components. |
| **FormField** | `form-field.tsx` | Form field wrapper. |
| **HoverCard** | `hover-card.tsx` | For sighted users to preview content available behind a link. |
| **Input** | `input.tsx` | Displays a form input field or a component that looks like an input field. |
| **InputContainer** | `input-container.tsx` | Wrapper/Container for inputs. |
| **InputOTP** | `input-otp.tsx` | Accessible one-time password component. |
| **Label** | `label.tsx` | Renders an accessible label associated with controls. |
| **Menubar** | `menubar.tsx` | A visually persistent menu common in desktop applications involved in complex tasks. |
| **MultiSelectFilter** | `multi-select-filter.tsx` | Filter component allowing multiple selections. |
| **NavigationMenu** | `navigation-menu.tsx` | A collection of links for navigating websites. |
| **Pagination** | `pagination.tsx` | Pagination controls. |
| **Popover** | `popover.tsx` | Displays rich content in a portal, triggered by a button. |
| **Progress** | `progress.tsx` | Displays an indicator showing the completion progress of a task. |
| **RadioGroup** | `radio-group.tsx` | A set of checkable buttons—known as radio buttons—where no more than one of the buttons can be checked at a time. |
| **Resizable** | `resizable.tsx` | Resizable panel groups and handles. |
| **ResizableTableHeader** | `ResizableTableHeader.tsx` | Table header with resize capabilities. |

| **Select** | `select.tsx` | Displays a list of options for the user to pick from—triggered by a button. |
| **SelectedItemCard** | `SelectedItemCard.tsx` | Card component for selected items. |
| **Separator** | `separator.tsx` | Visually or semantically separates content. |
| **Sheet** | `sheet.tsx` | Extends the Dialog component to display content that complements the main screen content. |
| **Sidebar** | `sidebar.tsx` | A sidebar component. |

| **Sonner** | `sonner.tsx` | An opinionated toast component for React. |
| **Switch** | `switch.tsx` | A control that allows the user to toggle between checked and not checked. |
| **Table** | `table.tsx` | A responsive table component. |
| **Tabs** | `tabs.tsx` | A set of layered sections of content—known as tab panels—that are displayed one at a time. |
| **TextFilter** | `text-filter.tsx` | A simple text input filter. |
| **Textarea** | `textarea.tsx` | Displays a form textarea or a component that looks like a textarea. |
| **Toggle** | `toggle.tsx` | A two-state button that can be either on or off. |
| **ToggleGroup** | `toggle-group.tsx` | A set of two-state buttons that can be toggled on or off. |
| **Token** | `token.tsx` | A token/chip component. |
| **Tooltip** | `tooltip.tsx` | A popup that displays information related to an element when the element receives keyboard focus or the mouse hovers over it. |
| **ValueHelpDialog** | `ValueHelpDialog.tsx` | A complex dialog for value selection/help. |
| **ValueHelpFilter** | `value-help-filter.tsx` | Filter component using value help. |
