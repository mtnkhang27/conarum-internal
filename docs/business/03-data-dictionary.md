# Data Dictionary (Plain Language)

The following dictionary defines the key entities and concepts used within the **Conarum Prediction** platform in plain, non-technical language.

| Entity | Description |
|--------|-------------|
| **Tournament** | A major sporting event (e.g., FIFA World Cup 2026, Premier League 2025/26). It has a defined start and end date and a specific format (e.g., knockout, league). |
| **Team** | A participating squad in a tournament (e.g., Argentina, Manchester City). Teams are identified by their name, three-letter abbreviation (TLA), and country flag. |
| **Match** | A single game between two teams within a tournament. It has a scheduled kickoff time, a venue, and a defined stage (e.g., Group Stage, Final). |
| **Player** | An individual employee participating in the prediction platform. Each player has a display name, an avatar, and a history of predictions. |
| **Match Outcome Prediction** | A player's choice for the result of a match (Home Win, Draw, or Away Win). Points are earned if the prediction is correct. |
| **Exact Score Bet** | A player's prediction for the precise final score of a match (e.g., 2:1). Correct bets result in cash payouts. |
| **Tournament Champion Pick** | A player's selection for the team they think will win the entire tournament. |
| **Leaderboard** | A ranked list of all players based on their total points earned from correct predictions. |
| **Match Weight** | A multiplier (e.g., 1x, 3x, 5x) assigned to a match based on its importance. Correct predictions for high-weight matches earn more points. |
| **Prediction Slip** | A temporary holding area for a player's selections before they are officially submitted to the platform. |
| **Bracket Slot** | A position in the tournament's knockout stage (e.g., Round of 16, Quarter-Final). It links two teams and potentially two matches (home and away legs). |
| **Tournament Status** | The current state of a tournament (e.g., Upcoming, Active, Completed, Cancelled). |
| **Betting Status** | The current state of betting for a tournament or match (e.g., Open, Locked). When locked, no new bets or changes are allowed. |
